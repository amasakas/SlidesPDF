# pdf-slide-viewer-generator
PDFスライド公開サイトを簡単に作成できるテンプレートレポジトリ

## 使い方
1. Fork または “Use this template”
2. GitHub PagesとGitHub Actionsを有効にする
3. [`assets/pdf`](assets/pdf/) にPDFをアップロードする
4. 完了✨ サイトが公開されます 🚀
